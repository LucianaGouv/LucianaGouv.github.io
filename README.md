***Disciplina:*** ECT3699 - DESENVOLVIMENTO WEB FRONTEND - T01 (2026.1)

***Projeto01: DESENVOLVIMENTO WEB FRONTEND***

Projeto desenvolvimento web frontend, onde o objetivo é criar uma página web simples utilizando HTML e CSS. O projeto inclui a criação de um arquivo HTML para estruturar o conteúdo da página e um arquivo CSS para estilizar a aparência da página. O projeto também envolve a inclusão de imagens e links para tornar a página mais interativa e visualmente atraente.
